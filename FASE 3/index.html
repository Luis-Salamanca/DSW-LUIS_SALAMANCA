Propósito

El presente sitio web tiene como objetivo principal de fomentar los objetos virtuales de aprendizaje frente a tématicas como Flex Box y CSS Grid.


Introducción

A través del paso del tiempo conforme a las necesidades las paginas web pasaron de ser estaticas a cada vez mas interacturar con el usuario, es asi como Flexbox y CSS Grid llegan a revolucionar; ofreciendo un diseño en sistema uni o bidimensional que facilita al desarrollador web la creacion en menos tiempo con diseños mas receptivos y que ademas funcionan en varios navegadores.
